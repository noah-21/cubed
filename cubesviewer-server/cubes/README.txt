You need a running Cubes server, configured and with data,
in order to use CubesViewer.

You can find an example Cubes projects with sample data in
this (forked) cubes-examples repository:

  * https://github.com/jjmontesl/cubes-examples

CubesViewer example views (src/htmlviews/) are based on the "webshop"
example, a fully working Cubes project including configuration
and data, which can be found in the aforementioned repository.

You can find more information about Cubes at :

  http://cubes.databrewery.org/

  and

  https://github.com/Stiivi/cubes
